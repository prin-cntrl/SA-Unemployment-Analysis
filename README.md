# SA-Unemployment-Analysis

## Overview
This project analyzes South Africa's unemployment trends using Python and SQL. It explores patterns, generates visual insights, and provides a simple dashboard.

## Technologies Used
- Python: Pandas, Matplotlib, Seaborn
- SQL: For data aggregation
- Tools: VS Code, CSV dataset

## Project Structure
- `notebooks/SA_Unemployment_Analysis.py` – Python program generating charts
- `sql/sa_unemployment_analysis.sql` – SQL queries
- `images/unemployment_trend.png` – Line chart of unemployment trends

## Objective
To demonstrate practical skills in data analysis, visualization, and querying real-world datasets.
